# data-visualisation
 Data Visualization Project

---

 About the Project

This project is about Data Visualization using Python.
The main aim is to understand data by creating different graphs and charts.
Visualization helps to easily identify patterns, trends, and relationships in the dataset.

---

Objectives

- Analyze the dataset
- Create visual charts and graphs
- Understand data trends
- Present data in a clear visual form

---

 Tools Used

- Python
- Pandas
- Matplotlib
- Seaborn
- VS Code / Jupyter Notebook

---

 Visualizations Created

- Bar Chart
- Line Chart
- Histogram
- Scatter Plot
- Pie Chart
- Heatmap
